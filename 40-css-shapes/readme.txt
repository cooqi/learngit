最主要是掌握，正方形、圆、三角形、梯形，其余的都是根据这些变形来的

如一个心形，就是两个圆角矩形（一边圆角一边直角）拼成的，五角星就是由两个钝角三角形和一个锐角三角形拼成的，等等，很多形状都是一个个拼成的
